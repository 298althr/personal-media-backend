# HasVoiceActivity

Voice activity detection availability flag returned by PMS.
PMS may return this as a boolean or as string values (`"0"` or `"1"`).



## Supported Types

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.HasVoiceActivityEnum`

```typescript
const value: shared.HasVoiceActivityEnum = HasVoiceActivityEnum.Zero;
```
